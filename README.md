# blog.mlcdf.fr

My personal website

## Install

```bash
python -V # should print 3.6+
python -m venv venv
source venv/bin/activate
python -m pip install -r requirements.txt
```

## Scripts

Start dev server on port 8000
```bash
./script/dev.sh
```

Build website
```bash
./script/build.sh
```

Build website for production
```bash
./script/publish.sh
```
